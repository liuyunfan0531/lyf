# 一、方案标题
智能宠物投食器
# 二、方案应用场景
主要用于宠物独自在家无人照顾，解决主人担忧
功能：1.通过wifi模块连入涂鸦网络平台进行管理
      2.可远程可手动定时、定量设置进行投喂操作
      3.可通过手机APP或者LCD屏查看喂食、余粮等信息记录
      4.对投喂剩余粮进行自动清理
      5.将涂鸦wifi摄像头接入wifi,通过APP可以视频监控查看
      6.设置固定插杆装置可以安装宠物喜欢的玩具
